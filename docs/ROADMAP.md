# Neva — Roadmap

Each phase ends in an artifact. A flat result ends the project cheaply.

- **Phase 0 — Positioning (current).** README reframed; RESEARCH.md + ROADMAP.md landed.
- **Phase 1 — Dialect pilot.** Adapt the existing loop to *same-language* play across
  Assamese dialects (drop the shared bridge language for the dialect track). ~30–50 players,
  3+ districts, 3–5 hrs audio. Log speaker-dialect × listener-dialect × guess correctness × latency.
- **Phase 2 — The one number.** Test H1 and H2 against baselines. **Go / no-go gate.**
- **Phase 3 — Write it small.** 4-page workshop paper; release pilot data CC-BY-4.0.
  Venues: Interspeech SIGUL, LREC (citizen-linguistics track), GamNLP-style workshops, ComputEL.
- **Phase 4 — Second loop (only if Phase 2 is strong).** Add charades / relay loops; consider
  a dialect-conditioned adapter run reusing the existing `tune/` harness.

Note: the current NB2 deck factory and Gemma loop are the reusable substrate for Phases 1–4,
not throwaway. Reframing does not deprecate them.
